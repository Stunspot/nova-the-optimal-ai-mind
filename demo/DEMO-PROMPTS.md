# Demonstration prompts

These prompts are designed to reveal behavior, not force canned wording. Record the actual model response and preserve any useful surprise.

For installation and a first success, begin with [`../START-HERE.md`](../START-HERE.md). This file is the reusable prompt reference after both plugins are discovered.

## 1. Tour

```text
Use $nova to take me on the interactive tour. I am curious but not ready to hand over a project or personal information.
```

Observable target: a warm, skippable invitation; four compact routes; no biography collection; clear memory and authority boundaries; a turn toward the user's real work.

## 2. Ludis character background

```text
Use $nova and $ludis-continuum. I need a background for a character: a royal cartographer who erased one island from every map and now hears its bells in dry land. Give me a playable past, two relationships, one dangerous truth, and an opening choice. Do not borrow a prebuilt setting.
```

Observable target: vivid but playable fiction; pressures and relationships that create choices; an opening move; no Port Zindra, Bracken, or other bundled campaign.

## 3. MIND composition

```text
Use $augment-of-mind. A small team has 48 hours to choose between a flashy demo with weak evidence and a quieter demo that proves the core claim. The founder wants spectacle, the operator wants reliability, and the audience is hesitant. Recommend a course, show the decisive uncertainty, and give us a communication plan that preserves everyone's agency.
```

Observable target: one integrated recommendation with assumptions, uncertainty, stakeholder fairness, timing, and voluntary adoption—not a list of Faculty monologues.

## 4. TestForge proof

```text
Use $software-verification. Review the verification package in this repository. Separate direct evidence, derived conclusions, residual risks, and human-only contest actions. Do not upgrade a claim merely because a file exists.
```

Observable target: evidence-class discipline, executable or inspectable checks, explicit residual risk, and no claim that external submission events were observed.

## 5. Direct-first negative control

```text
Use $nova. Rewrite “meeting moved to four” as a polite one-sentence message.
```

Observable target: one polished sentence. No specialist narration, research, MIND coalition, memory operation, or game machinery.
