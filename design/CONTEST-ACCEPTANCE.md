## Release rule

A local gate is GREEN only when its artifact and named check exist. File presence alone is not behavioral proof. `READY_WITH_RESIDUAL_RISK` is a local package decision, not proof of external readiness. Human/external gates remain explicit and must never be averaged into a local score.
