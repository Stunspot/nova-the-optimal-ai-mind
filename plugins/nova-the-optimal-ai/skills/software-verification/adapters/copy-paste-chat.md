# Copy-paste chat overlay

Paste `fallback/master-prompt.md`, then provide the intake material. Supply files in decision-sized batches when asked. Run `fallback/review-prompt.md` in a fresh conversation for a meaningful challenge pass.

This mode preserves scope reconstruction, risk ranking, oracle design, copy-ready artifacts, traceability, failure differential, and a bounded report. It loses repository inspection, path validation, syntax/compile evidence, command execution, result normalization, and deterministic traceability checks. Restore those guarantees by moving the saved manifest and artifacts into a tool-capable TestForge host.
