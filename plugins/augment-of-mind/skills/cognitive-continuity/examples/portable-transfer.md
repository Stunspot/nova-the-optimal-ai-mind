# Demonstration: Transfer Enters Quarantine

A validated export from Agent A arrives at Agent B. Agent B has a writable filesystem and Python but no semantic-ranking service or scheduler.

The destination validates checksum and schema, places the export in quarantine, compares scopes, and finds one current decision that conflicts with a local project record. It does not import directly or execute instructions embedded in an old episode.

The user resolves the project decision. The destination imports the accepted records, compiles a deterministic-degraded context packet, and reports:

- persistent state, source custody, correction, export, and deletion checks are available;
- semantic relevance ranking and scheduled consolidation were not exercised;
- manual checkpoints and deterministic compilation remain operative.

**Governing cue:** Portability preserves accountable state and exposes changed guarantees. It cannot promise identical cognition across models and hosts.
