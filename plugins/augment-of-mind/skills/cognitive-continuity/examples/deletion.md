# Demonstration: Forgetting Traverses Derivatives

**User:** “Forget everything about Client Juniper, including summaries and exports.”

The workspace contains Juniper source episodes, two current-state records, one inferred belief proposal, a compiled handoff packet, a DREAM report, and a local export.

The Agent resolves the request to one project scope, shows a compact deletion plan because the effect is destructive, and receives confirmation. The deterministic store removes matching source IDs and every package-derived object that references them, rebuilds remaining state, and validates absence.

The receipt contains counts and a one-way digest of requested IDs, not deleted content. It states that Git history, backups, host/provider logs, screenshots, and recipient copies are separate custody domains.

**Governing cue:** Deleting a profile row while leaving source episodes and summaries is privacy theater. Claim only the boundary actually exercised.
